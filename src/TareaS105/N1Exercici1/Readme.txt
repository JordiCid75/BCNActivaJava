Para ejecutar este programa desde ínea de comandos se deben hacer las siguientes instrucciones

1- Copia los ficheros y la estructura del proyecto en la carpeta que desees. (Ejemplo: c:\Proyecto\
2- Abre un terminal y situate en la carpeta del punto anterior (Ejemplo: c:\Proyecto>)
3- ejecuta la compilación con el siguiente comando:
	javac ./TareaS105/N1Exercici1/*.java
4- ejecuta la aplicaicón usando el siguiente comando pasándole por parametro la carpeta que quieres examinar:
	java -cp . TareaS105.N1Exercici1.main //Users/jordicid/documents/treb
